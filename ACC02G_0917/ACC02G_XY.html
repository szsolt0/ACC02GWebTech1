SZÓKIRAKÓ JÁTÉK

Adott egy 4*99-es négyzetháló. A program megad egy - a négyzetháló feletti képhez kapcsolódó - szót vagy mondatot, majd ennek betűit összekeverve elhelyezi ebbem a négyzethálóban.

A játékos feladata az, hogy az összekevert betűkből kirakja a szót

vagy mondatot a kijelölt helyen. A betűket vísszintes vagy függőleges irányban egyenként lehet mozgatni.

Ha a betű útját elállja egy másik betű, akkor az csak a másik betűig tolódik el.

Miután sikerült a betűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely betűketfelcseréltük, akkor azokra színkiemeléssel hívja fel a figyelmünket.

a2 + b2 = c2

Általános háromszög esetén is igaz:
α + β + γ = 180°
