legfontosabb git parancsok:

  • git add: hosszóad fileokat és jegyzékeket a staging-hez később lehessen commit-olni őket. A -A mindent hozzáad.
  • git commit: commit-olja azokat a fileokat és mappákat amiket hozzáadtunk a staging-hez. A -m<message> kapcsolóval lehet egyből commit üzenetet írni, de ha kihagyjuk akkor megnyílik az alapértelmezett szövegszerkeztő amibe lehet írni.
  • git push: szinkronizálja a local repsitoryt a remote repositoryval
  • git pull: szinkronizálja a remote repsitoryt a local repositoryval
  • git status: információ a jelenlegi repóhoz. pl. milyen fileok lesznek commitolva egy git commit esetén, milyen branch-en vagyunk jelenleg, etc.
  • git init: repository létrehozása
  • git checkout <branch>: átváltás egy branch-be. A -b kapcsoló újat hoz létre
  • git branch -d <branch>: branch törlése
  • git stash: elmenti a munkajegyzék jelenlegi állapotát. A git stash pop visszaállítja
  • git remote add: remote repository hozzáadása
  • git merge <branc>: A megadott összefűzése a jelenlegi branch-el.
